## im-select

| Key        | Map to                                    |
| ---------- | ----------------------------------------- |
| `CapsLock` | `Ctrl`                                    |
| `Ctrl+[`   | `Esc`                                     |
| `Esc`      | `Esc` && switchIMEConversionMode("EN")    |
| `LShift`   | `LShift` && switchIMEConversionMode("ZH") |

## 2. Raycast Navigation binding

| Key      | Map to   | Desc             |
| -------- | -------- | ---------------- |
| `Ctrl+o` | `Ctrl+K` | actions          |
| `Ctrl+h` | `Left`   | navigation left  |
| `Ctrl+j` | `Down`   | navigation down  |
| `Ctrl+k` | `Up`     | navigation up    |
| `Ctrl+l` | `Right`  | navigation right |
